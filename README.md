## What is Fasyra?

Fasyra is an AI-powered platform that helps businesses automate customer service through intelligent virtual assistants.

The platform allows companies to create customized AI agents capable of answering questions, generating quotes, qualifying leads and integrating with websites, WhatsApp and social media.
